Terrain, Watershed & Visibility Analysis Using Raster GIS

Terrain Modeling & Hydrologic Analysis | Sept 2025

Technical Implementation

  • Conducted multi-resolution terrain analysis using LiDAR-derived and legacy DEMs to assess elevation differences, generate hillshades, and evaluate the impact of raster resolution on surface representation and derived analyses.
  • Performed visibility modeling by computing viewsheds and line-of-sight paths from specified observer locations, distinguishing visible and obstructed terrain using geodesic viewshed and line-of-sight tools.
  • Implemented a complete hydrologic workflow—including pit filling, flow direction, flow accumulation, snap pour point, and watershed delineation—to derive stream networks and watershed boundaries from elevation data.
  • Generated elevation profiles along transportation corridors by interpolating elevation values onto vector paths and exporting profile graphs to quantify slope and terrain variability along routes.
